Guide: Ubuntu Server Installasjon og Webserver Oppsett

Oppsett av Virtuell Maskin

1. Høyreklikk på din pool og velg "New Virtual Machine"

2. Gå gjennom veiviseren:

Velg din egen pool og klikk next. På nummer 4 velger du IKT-fag og klikker next. På nummer 5 klikker du bare next.

3. Velg Linux og Ubuntu Linux (64-bit)

4. Velg RAM og harddisk størrelse. Husk å velge "Thin Provision" for "New Hard disk"

5. Velg ubuntu-22.04.1-live-server-amd64 fra datastore ISO og huk av "Connect at power on"

6. Aktiver "Force EFI setup" i Boot options

Ubuntu Server Installasjon

Start VM-en og klikk på "Launch Web console"

Trykk Enter for å starte installasjonen

For nettverkskonfigurasjon, velg manual og fyll inn:

  • Subnet: 10.12.xx.0/24
  • Address: 10.12.xx.xx
  • Gateway: 10.12.xx.1
  • Name Servers: 10.10.1.20, 1.1.1.1, 8.8.8.8

Installer OpenSSH server

Når installasjonen er ferdig, velg "Cancel update" eller "Reboot now"

SSH Oppsett

Koble til via terminal:

ssh [username]@10.12.XX.XX

Hent din public SSH-nøkkel:

cat ~/.ssh/id_rsa.pub

Apache Installasjon

Oppdater systemet og installer Apache:

sudo apt update && sudo apt upgrade -y
sudo apt install apache2 -y
sudo apt update && sudo apt install git -y

Clone GitHub-prosjektet og sett rettigheter:

sudo git clone [git_repo_link]
sudo chown -R www-data:www-data /var/www/html/myProject
sudo chmod -R 755 /var/www/html/myProject
